# Gerenciador de Tarefas
Aplicação web para gerenciamento de tarefas com ordenação por prioridade e prazo.

## Funcionalidades
- Criar novas tarefas com título e data
- Marcar tarefas como importante
- Marcar tarefas como completas
- Remover tarefas
- Ordenação automática por prioridade e prazo.

## Tecnologias
- React
- CSS Modules

## Como executar
1. Clone o repositório
2. Instale as dependências usando o comando npm install
3. Rode o projeto: npm run dev
